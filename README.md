# Breast Cancer Prediction using Machine Learning

This repository contains a machine learning project focused on predicting whether a tumor is Malignant or Benign based on diagnostic data. The project involves data preprocessing, exploratory data analysis (EDA), and the implementation of classification algorithms to achieve high accuracy.

### Project Overview

The goal of this project is to provide a reliable classification tool to assist in the early detection of breast cancer. Using the Wisconsin Breast Cancer Diagnostic Dataset, we analyze various cell nuclei characteristics to build a predictive model.

### Features

* Data Cleaning: Handling missing values and encoding categorical labels.

* EDA: Visualizing feature correlations and distributions.

* Model Training: Comparison of multiple classifiers (e.g., Logistic Regression, Random Forest, SVM).

* Evaluation: Detailed metrics including Accuracy, Precision, Recall, and F1-Score.

### Tech Stack

* Language: Python

* Libraries: Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn

* Notebook: Jupyter Notebook / Google Colab
